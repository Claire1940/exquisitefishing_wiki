<Callout type="tip" title="Prioridade de Gastos">

</Callout>

O **Anzol Ômega** é o item mais caro do jogo, mas é necessário para a cena final. Recomenda-se pescar repetidamente o Tubarão do Oceano Turbulento (900m) para acumular os fundos necessários para esta compra final.

<Section icon={Trophy} title="Guia Passo a Passo para Conclusão de Cenas">

Siga estes passos para desbloquear eficientemente cada cena do jogo sem desperdiçar ouro em itens desnecessários no início do jogo. Esta rota foca em maximizar o ouro por lançamento para alcançar o fim do jogo o mais rápido possível.

<Steps>
  <Step num="1" title="A Base no Oceano Calmo">
    Comece pescando a Sereia (50m, Anel de Diamante) para ganhar seus primeiros milhares de ouro. Assim que tiver o suficiente, compre o Anzol Bom e a Calcinha Listrada para pescar o Tubarão (200m). Use esse dinheiro para comprar a Caixa de Chocolates e pescar o Peixe-Lanterna (300m) até ter 1.500g para o Barco Regular.
  </Step>
  <Step num="2" title="Expansão para o Oceano Rochoso">
    Entre no Oceano Rochoso e mire imediatamente no Polvo (100m) usando o Anzol de Escudo Duplo e o Consolador Poderoso. Seu objetivo principal aqui é o Peixe-Lanterna de 600m (Anzol Reforçado + Rosa Vibrante), que paga 13.000g. Pesque este alvo repetidamente para economizar 5.000g para o Barco Reforçado.
  </Step>
  <Step num="3" title="Profundidades Turbulentas e Itens de História">
    No Oceano Turbulento, foque no Tubarão a 900m (Super Anzol + Anel de Diamante) para obter 24.000g por captura. Este é o melhor ponto de acúmulo de ouro no meio do jogo. Certifique-se de mergulhar até 1.000m pelo menos uma vez para coletar a **Carta do Vovô**, que é um colecionável crítico da história.
  </Step>
  <Step num="4" title="O Final no Oceano Escuro">
    Compre o Barco Dourado (8.000g). Compre o Anzol Plug Anal e o Anzol de Contas Anais para finalizar as cenas do Kappa e do Polvo. Por fim, economize para o Anzol Ômega para capturar o Peixe-Lanterna de 2.000m. Isso completará sua galeria e desbloqueará a interação final de personagem.
  </Step>
</Steps>

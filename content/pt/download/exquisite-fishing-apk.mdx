<Section icon={Trophy} title="Localização dos Arquivos de Save e Guia de Backup">

Se você estiver jogando a versão para PC ou transferindo o progresso do **exquisite fishing apk**, é altamente recomendável manter um backup regular do seu arquivo de save para evitar a corrupção de dados.

### Caminho do Save no Windows PC
Nos sistemas operacionais Windows, seus dados de salvamento local são armazenados na pasta oculta AppData. Você pode acessá-la diretamente copiando o seguinte caminho no seu explorador de arquivos:

`%USERPROFILE%\AppData\LocalLow\PinkySoul\Exquisite Fishing`

### Dicas de Gerenciamento de Save
- **Backups Manuais**: Copie toda a pasta `Exquisite Fishing` para uma unidade externa antes de atualizar o cliente do jogo.
- **Migração de Progresso**: Para mover seu progresso para um dispositivo diferente, copie os arquivos de salvamento exatamente para a mesma estrutura de diretório em sua nova máquina.
- **Correção de Corrupção**: Se o jogo não carregar, exclua o arquivo de save corrompido neste diretório e substitua-o pelo seu arquivo de backup.
